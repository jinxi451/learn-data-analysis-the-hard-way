## 练习环境

1. 安装 Docker 
2. 克隆项目 并运行下面的命令:
```banh
cd docker_project/learn-data-analysis-the-hard-way/
cd SQL/interview/docker/
```
3. 运行下面的命令:
```bash
docker-compose up
```

4. 打开浏览器 输入`localhost:8080`

参照输入下面信息 用户密码可在 `docker-compose.yml` 中找到为`rootpassword`
